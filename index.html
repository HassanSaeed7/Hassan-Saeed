Link Keeper

  • Browser Extension App
  • Client-Side
  • Local Storage

Track any leads, links, research articles, etc. with this chrome extension that I completed while following the Scrimba JavaScript tutorials. The extension saves the link you input in the text field or it saves the current tab's link in your browser's local storage. This way, your list of links is retained between browser sessions.

Originally, the Scrimba tutorial did not teach us how to program a download button and I felt this function was necessary to make the extension a little more robust, so I researched a method to add a download function. Now, you can share your leads (or list of links) with your team even faster with a .csv file.
